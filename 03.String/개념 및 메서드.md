# 문자열
자바스크립트는 문자 자료형은 존재 하지 않고, 문자열 자료형만 존재한다.

<br>
자바스크립트에서 문자열은 문자의 배열이 아니다.
<br>
배열은 가변 값이지만 문자열은 불변 값이다.
<br>

문자열은 불변 값이므로 문자열 메소드는 항상 새로운 문자열을 생성한 후 반환한다.
